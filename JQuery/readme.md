# Meeple Garage - E-commerce HTML Template

Este é um projeto de e-commerce onde aplico conhecimentos adquiridos sobre JQuery utilizando HTML, CSS, Bootstrap, Owl Carousel e jQuery. 167 

## Funcionalidades

- **Navbar Responsiva**: Uma barra de navegação responsiva usando Bootstrap que permite aos usuários navegar entre as páginas do site.
- **Carrossel de Produtos**: Um carrossel de produtos em destaque implementado com Owl Carousel e jQuery, exibindo os jogos disponíveis com imagem, nome e preço.
- **Design Responsivo**: Utiliza Bootstrap para criar um layout responsivo, garantindo que o site fique bem em diferentes tamanhos de tela.
- **Rodapé com Links e Ícones Sociais**: Um rodapé que inclui links para páginas importantes do site e ícones de redes sociais usando Font Awesome.
- **Interatividade com jQuery**: jQuery é utilizado para adicionar interatividade, como o funcionamento do carrossel e a manipulação do DOM.

## Tecnologias Utilizadas

- **HTML5**: Estruturação da página.
- **CSS3**: Estilização personalizada.
- **Bootstrap**: Framework CSS para responsividade e componentes prontos.
- **Owl Carousel**: Plugin jQuery para carrossel de imagens.
- **Font Awesome**: Ícones vetoriais.
- **jQuery**: Biblioteca JavaScript para manipulação de elementos da página.

## Como Executar o Projeto

1. Faça o download ou clone o repositório.
2. Certifique-se de que todos os arquivos necessários (CSS, JavaScript, imagens) estão na estrutura de diretórios correta.
3. Abra o arquivo `index.html` em um navegador web para visualizar o site.
